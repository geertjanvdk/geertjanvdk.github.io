MySQL/Sun/Oracle Era

  • MySQL Router is a tool with which, among other things, you can redirect incoming connections to well defined MySQL Servers. The Connection Routing plugin, for example, can be handy to redirect applications to one of the read-only slaves. MySQL Router is written using C++11.

  • MySQL Connector/Python is Python DB API v2 compliant driver, implementing the MySQL protocol pure Python as well as linking with MySQL C Client Libraries. The project is owned by Oracle, but I started coding on this while employed at MySQL AB.
