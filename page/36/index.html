Disabling binary logging when restoring a MySQL dump

Fri Jul 17, 2009
mysql

There is a cool option for mysqlbinlog for disabling the binary log when doing recovery using binary logs, namely --disable-log-bin. How, one would think it is also avialable for something like mysqldump or even the mysql CLI? Nope. There are various ways for doing this, here is one: $ (echo "SET SESSION SQL_LOG_BIN=0;"; cat dump.sql) > dump_nobinlog.sql Obviously, bit pain for really big files, and when dumping to multiple files.
