Netways Nagios Conference 2008

Thu Sep 11, 2008
mysql / community / nagios

The Netways Nagios Conference 2008 in Nürnberg (DE) was opened today by Ethan Galstad, creator of Nagios. He gave an overview of how Nagios evolved and the steady growth of the monthly downloads, now reaching 16k! The Nagios 3.0 overview was quite nice. Apparently lots of effort was put in optimizing it for bigger setups, like doing host checks in parallel and changing internal structures not using linked lists. Great was also to hear that the web interface is being revamped, as well as the Embedded Perl becoming some kind of plugin, called Event Broker Modules.
